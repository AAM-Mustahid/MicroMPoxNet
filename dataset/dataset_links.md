# Dataset Resources

This repository uses two datasets for training and evaluation.

---

## 1. Mpox2025 Dataset (Proposed)

The **Mpox2025 dataset** was constructed to support reliable Monkeypox classification using dermoscopic images.

Dataset characteristics:

- Total images: **696**
- Mpox images: **337**
- Normal images: **359**
- Image format: **PNG**
- Color space: **RGB / Grayscale**

After applying augmentation techniques, the dataset was expanded to **34,555 images** for training experiments.

Dataset Download:

https://drive.google.com/file/d/1P3ZZ01TSJS4v9uhBy0USzfbMXWbO8QIh/view?usp=sharing

---

## 2. External Dataset (Multiclass Classification)

A public dataset from Kaggle was used for multiclass classification experiments.
